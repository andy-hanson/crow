Compiling to JavaScript

Most Crow code can be compiled to JavaScript.
The exception is any code relying on extern native or any other extern library. That will be explained in {{tutorial/extern}}.

import crow/js: call, js-global main void() trusted, js extern _ = "alert".js-global call "Hello, JS!"::string info log "This goes to log below"
On this page, examples will run in the browser instead of using the interpreter. Logging will still be redirected to output though.

Currently, there's no way to generate imports. If you want to use a third-party JS library, the easiest way is to write it to a global that Crow can access.
