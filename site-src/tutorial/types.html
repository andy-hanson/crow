bool

There isn't much to say about bool;
you can create one with true or false and use it with if.

false can also be written as (), so it can be an implicit else.

Unlike in other languages, bool is not a number type, and numbers aren't treated like booleans.

Common functions on bool are in the crow/bool module.

number types

Types of numbers are:
nat8, nat16, nat32, nat64 Natural numbers, in the range 0 to 2n - 1.
int8, int16, int32, int64 Integers, in the range -2n - 1 to 2n - 1 - 1.
float32, float64 Floating-point values.

There are aliases nat for nat64, int for int64, and float for float64.

main nat^(_ string[]) summon a nat = 3 info log "{a / 2}" b int = 3 info log "{b / -2}" c float = 3 info log "{c / 2}" info log "{a.to::int64 + b}" ()

Arithmetic functions work on values of the same type only, so you can add nat + nat or int + int, but not nat + int. To add a nat and int, you must convert one to the other's type.

Arithmetic functions throw exceptions on overflow (or underflow).

main nat^(_ string[]) summon a nat8 = 255 info log "{a + 1}" ()

Common functions on numbers are in the crow/number module.

string and char

We've been using string for most of this tuturial so far.
A string is sequence of chars.
A char is an 8-bit value, so a Unicode character might use multiple.

Unlike in other languages, char is not a number type, and string is not just a char[]; though conversion functions are available.

"" can also be written as (), so it can be an implicit else.

Functions for string and char are in the crow/string module.

symbol

symbols are unique values identified by a string.
(Other languages may call these interned strings.) main nat^(_ string[]) summon x symbol = "flabbergast" info log "{x}" info log "{x == "glabberfast"}" info log "{x == ("flabber" ~~ "gast").to}" ()

The syntax for a symbol is the same as for a string.

Unlike a string, you can't manipulate the contents of a symbol or use string interpolation (except by converting to/from string).

When a variable is a symbol, that indicates that it should belong to a small set of possible values, and will be compared against other values in the set.

== is O(1) for symbol and O(n) for string.

Functions for symbol are in the crow/symbol module.
