Currently, the language only works on 64-bit Posix systems. If your system is not Linux and x64, you'll have to build from source.

Linux x64

Click the above link to download, then run:

cd ~/Downloads
tar -xJf crow.tar.xz
cd crow
bin/crow run demo/hello

This should print Hello, world!

Installing

Optionally, add this to your ~/.bashrc (or equivalent):

export PATH="~/Downloads/crow/bin:$PATH"

to make crow available on the command line anywhere.
