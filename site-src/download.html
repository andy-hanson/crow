  • Linux x64
If you want it ported to a different system, just ask.

After downloading, test it out with:

tar --extract --xz --file ~/Downloads/crow.tar.xz
./crow/bin/crow run crow/demo/hello
This should print hello, world!.

If there was an error, make sure libgccjit is installed on your system.

To install:

sudo ln --force --symbolic $PWD/crow/bin/crow /usr/local/bin/crow
crow run crow/demo/hello
