You can run it by clicking the play button above, or download and run crow hello.crow. This shows a few basic features of noze:
  • Modules: The main module imports types and functions from io.
  • A function: main. The name of a declaration always goes first. So you'll have r record, u union, and here, main fut exit-code.
  • Templates: fut and arr are templated types, and resolved is a templated function. The type arguments to resolved are inferred from the return type of main).
  • IO: The main function is labelled summon because it calls print which is also summon. As we'll see, most code will actually not do I/O this way.
