Installation

Noze has the following dependencies:
  • DMD: install version 2.093 or greater.
  • SDL2: Install this from your package manager.
Then to install Noze:
  • Download the repo (TODO: HOW?)
  • Run ./build to build.
  • You may add the bin directory to your path, or just write the path explicitly.

Types

Noze has only a few kinds of types:
  • builtin: Things like bool that can't be expressed in terms of other types.
  • extern-ptr: A pointer to data from an external library that can only be used through that library.
  • record: Type of a value that contains several named values. An example would be a point with x and y fields.
  • union: Type of a value that is one of several possibilities. An example would be a shape with circle or square choices.
record and union types can each have primitive operations for using them. builtin and union types can only be used through certain functions and are otherwise inscrutable.
